

Alla textnummer i NiKom globala. Detta gör det väldigt lätt att flytta
texter mellan möten om tex om diskussionen spårar ur. Man ger då kommandot
'!flytta <mötesnamn>' i radeditorn. (Qtrl-Q, Flytta <möte> i
fullskärmseditorn. Kommentarslänkningen behålls dock fortfarande och man
får läsa kommentarer även om de är flyttade förutsatt att man är medlem i
det mötet som texten är flyttad till.

Om man vill kan man pausa utskriften med mellanslag, för att kunna göra
detta ska man slå på flaggan 'Mellanslag som paustangent'. Man kan dock
alltid pausa utskriften med Ctrl-S.
Man kan även välja om man vill få fram <RETURN> lite då och då. Om man
sätter 'Antal rader:' i sin status till 0 så får man inte det.

Alla textfiler som man kommer åt med Info-kommandot går att förkorta till
sin närmast unika del. Istället för att skriva 'info Generallistan' så
räcker det tex utmärkt med 'inf gen'.

Man kan definiera en egen profile, dvs en sorts startup-sequence, som körs
automatiskt när man loggar in. I den kan man ha vanliga NiKom-kommandon typ
'lista nyheter'. De kommandon som finns är: 'Ändra Profile', 'Visa Profile'
och 'Exekvera Profile'.

Man kan skriva en lapp som visas när man tar status på en person. Den
skriver man med kommandot 'Ändra Lapp'. 'Radera Lapp' tar bort den. Om man
vill att denna lapp ska komma upp när någon skriver brev till en  så kan
man slå på flaggan 'Lapp vid brev'.

Om man vill ha mer information om ett kommando kan man använda 'Manual'-
kommandot. Tex 'Manual lista möten' ger en beskrivning på kommandot 'Lista
Möten'.

NiKom har en fullskärmseditor. För att använda denna måste
Fullskärmseditor-flaggan slås på. För mer info om fullskärmseditorn finns
det en info-fil.

Om man vill kan man definiera egna alias som man kan använda i basen. Tex
om man tycker det är väldigt besvärligt att skriva 'lista filer' så kan man
definiera ett alias så man bara behöver skriva 'f' istället. Se
manual-filen för kommandot 'Alias' för mer information.
